## 📌 Project Overview

Credit Risk Assessment is a Machine Learning project that predicts whether a loan applicant is likely to be **High Risk** or **Low Risk** based on financial and personal information. The project uses a **Random Forest Classifier** to analyze applicant data and assist financial institutions in making informed loan approval decisions.

---

## 🎯 Project Objective

The main objective of this project is to build a classification model that can accurately predict the credit risk of loan applicants using historical financial information. This helps reduce loan default risk and supports smarter lending decisions.

---

## 📊 Dataset Information

The dataset contains applicant information with the following features:

| Feature | Description |
|----------|-------------|
| Applicant_ID | Unique ID of each applicant |
| Income | Applicant's income |
| Credit_History_Score | Credit score of the applicant |
| Employment_Status | Employment status (Employed, Self-Employed, Unemployed) |
| Loan_Amount | Requested loan amount |
| Risk_Label | Target variable (High Risk / Low Risk) |

## 🚀 Installation

Clone the repository:

