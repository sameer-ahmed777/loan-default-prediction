# 💳 Credit Risk Assessment using Machine Learning

## 📌 Project Objective

The objective of this project is to develop a Machine Learning model that predicts the credit risk of loan applicants. By analyzing important applicant information such as income, credit history score, employment status, and loan amount, the model classifies applicants into **High Risk** or **Low Risk** categories. This helps financial institutions make informed lending decisions and reduce the risk of loan defaults.

---

## 📊 Dataset Information

The dataset contains applicant information with the following features:

| Feature | Description |
|---------|-------------|
| Applicant_ID | Unique ID of each applicant |
| Income | Monthly or annual income of the applicant |
| Credit_History_Score | Applicant's credit score |
| Employment_Status | Employment status (Employed, Self-Employed, Unemployed) |
| Loan_Amount | Requested loan amount |
| Risk_Label | Target variable (High Risk / Low Risk) |
